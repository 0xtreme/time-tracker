# Privacy and Terms

## Privacy

This version stores time tracking data only in the user's browser through `localStorage`.

The app does not send session data to a server, does not require login, and does not include analytics or third-party tracking.

Users are responsible for exporting backups if they need long-term retention. Clearing browser data, using private browsing, or changing devices can remove local records.

## Optional Future Accounts

If cloud sync, Google sign-in, GitHub sign-in, or other account features are added, this document must be updated before release. A future account-based version should clearly disclose what data is stored, where it is stored, how it can be exported, and how it can be deleted.

## Terms

This tool is provided as-is for manual time recording. Users should review copied times before submitting them to a client, employer, payroll system, or project management tool.

The app does not guarantee that a recorded session represents billable time. It records the timestamps entered or produced by user actions in the browser.
